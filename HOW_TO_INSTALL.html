🏥 How to Install MedCoreX on Android

✅ MedCoreX is a PWA (Progressive Web App) — it works exactly like a native app on your Android phone. No Play Store needed!

📱 Method 1 — Using Chrome (Recommended)

1

Copy files to a server or open locally

Put all the MedCoreX files (index.html, manifest.json, sw.js, icons/ folder) together in one folder. You can host them on any web server, or use a local server app on Android.

2

Open Chrome on your Android phone

Open Google Chrome browser and navigate to your MedCoreX URL (e.g. http://yourserver.com/medcorex/)

3

Tap the Install banner

A banner will appear at the bottom saying "Install MedCoreX App" — tap the Install button. Or tap the Chrome menu (⋮) → "Add to Home screen"

4

Confirm installation

Tap "Install" in the popup. MedCoreX will be added to your home screen with the hospital icon, just like any other app!

5

Open from Home Screen

Tap the MedCoreX 🏥 icon on your home screen. It opens in full screen (no browser address bar) — looks and feels like a real app!

🌐 Easiest Way to Host (Free)

💡

Use GitHub Pages (Free hosting)

1. Create a free GitHub account at github.com
2. Create a new repository
3. Upload all MedCoreX files
4. Go to Settings → Pages → Enable GitHub Pages
5. Your app will be live at https://yourusername.github.io/medcorex/

⚠️ Important: PWA install prompt only appears when the app is served over HTTPS (not http://). GitHub Pages and Netlify both provide free HTTPS automatically.
🔒 Your data stays private — all patient and doctor data is stored locally on the device using browser storage. Nothing is sent to any server.

✅ What works offline?

Once installed, MedCoreX works fully offline — you can add records, view patient data, write prescriptions, and print reports without internet connection.
